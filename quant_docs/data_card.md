# 数据卡
- 数据源：OKX 永续合约公开 K 线（ccxt 获取）
- 供应商/交易所：OKX Mainnet（历史数据不依赖 API Key）
- 覆盖时间：训练与回测脚本常用 `2020-01-01` 至最近（以配置为准）
- 频率与时区：默认 `1d`；时区/收盘定义需以 OKX K 线口径确认为准
- 字段定义：Open / High / Low / Close / Volume

## 1. 质量与清洗
- 缺失处理：由 `DataValidator` 校验与修复（`crypto_trader/data_validator.py`）
- 异常值处理：当前无显式异常值剔除逻辑，主要依赖交易所数据质量

## 2. 版本与可复现
- 数据版本号：按下载时间与区间记录（建议在 `experiment_log.csv` 中落库）
- 存储位置：运行时通过 ccxt 拉取，未持久化版本快照

## 3. 加密特有字段
- Funding/Mark Price：当前未在数据源中显式使用，仅在回测成本模型中以常数近似
- 手续费/费率：回测使用常数成本，实盘以交易所实际费率为准
